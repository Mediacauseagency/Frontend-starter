data-pretty-number

This function will loop through all elements with the data-pretty-number attribute and format and display the attributes into the element as shown below. To use it, just require the helpers/prettyNumbers file.

data-toggle-classes-self

This function will loop through all elements with the data-toggle-classes-self attribute and add a click event to it to toggle the given classes. The click event will prevent default unless the event target has an href other then a hash.

data-toggle-classes-target

This function will loop through all elements with the data-toggle-classes-target attribute and add a click event to it to toggle the given classes on a target. The click event will prevent default unless the event target has an href other then a hash. The first argument in the attributes should be the target's selector and the following arguments should be the class names.
